It's a new month, but the wallpaper hasn't changed!

The wallpaper isn't always immediately made available at the beginning of each month. It sometimes takes a few days for the Logos Hope people to upload the new wallpaper. (I'm sure they're busy people!) You can check the website to make sure.

The wallpaper changed, but most of it is grey!

This can happen if the wallpaper file didn't download properly. Since the file is already downloaded, clicking Set Wallpaper will simply set the wallpaper to the already-downloaded file, which won't solve the problem. In this case, you need to click the Clear Cache button below. That will remove the damaged file. Clicking Set Wallpaper will then download the wallpaper again.

I cleared the cache and downloaded the wallpaper again, but the wallpaper didn't change!

Since the name of the file being set as the wallpaper hasn't changed, the system may not bother reading the file again. (It assumes it hasn't changed.) Try setting your wallpaper to something else, then clicking Set Wallpaper again.
